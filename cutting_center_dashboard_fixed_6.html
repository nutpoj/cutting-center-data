💡 ทำงานข้ามเครื่อง: Export .json จากเครื่องนี้ → ไป Import บนเครื่องใหม่ → ข้อมูลทั้งหมด (เป้า, OEE, ยอด, สี, ลำดับ) จะตรงกัน
⚠️ ถ้าเครื่องใหม่มีข้อมูลเก่าค้าง → กด "🗑️ ล้างข้อมูล" ก่อนแล้ว Import เพื่อความสะอาด

⬆️ ผลการ Import

❓ คู่มือการใช้งาน — Import ข้อมูล คลิกเพื่อ ขยาย/ซ่อน
📋 รองรับไฟล์ 3 ประเภท

📘 CuttingCenter_BACKUP.json — ไฟล์สำรองจากโปรแกรม (โอนย้ายเครื่อง)
📙 CuttingCenter_YYYY-MM-DD.xlsx — ไฟล์ Export จากโปรแกรม

🔢 ขั้นตอน (รายวัน)
① รับไฟล์ CheckPart.xlsx จากแผนก IT / ระบบ Part Computer
② กดปุ่ม ⬆️ Import → เลือกไฟล์ → รอผลการ Import
③ ตรวจสอบผลใน popup "ผลการ Import" — ดูว่ามี error หรือไม่
④ หลัง Import เสร็จ → Export .json สำรองข้อมูลทันที

⚠️ Import ซ้ำวันเดิมจะ merge ข้อมูล ไม่ทับ · Taiwan ใส่ยอดเองผ่านปุ่ม ✏️ ใส่ยอด
📌 รูปแบบไฟล์ที่รองรับ (ปุ่ม Import เดียวกัน)
📗 CheckPart.xlsx — ไฟล์ยอดผลิตจากระบบ (มีคอลัมน์ Part Computer, Code Part)
📘 CuttingCenter_BACKUP.json — ไฟล์สำรองจากโปรแกรม (เครื่องจักร+เป้า+ยอดผลิต)
📙 CuttingCenter_YYYY-MM-DD.xlsx — Export จากโปรแกรม (ยอดผลิต+เป้าหมาย)
⚠️ สำคัญ: ถ้าเครื่องใหม่เห็นข้อมูลเดิมก่อน Import → เป็นข้อมูลตัวอย่างที่ฝังในไฟล์ HTML
